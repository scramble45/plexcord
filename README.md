# Configuration

## Environmentals:

- `discord_token=""`
- `auth_password="password"`
- `plexLibrary="./com.plexapp.plugins.library.db"`
- `external_hostname="somehostname.com"`

## Startup:

Development:
> `npm run dev`

Production:
> `npm start`

## Commands:

Prefix:
> `~!`

Example command:
> `~!help`
