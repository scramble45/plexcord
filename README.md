# PlexCord
a discord bot that can query a plex db

## Configuration

## Environmentals:

- `discord_token=""`
- `auth_password="password"`
- `database="./com.plexapp.plugins.library.db"`
- `external_hostname="somehostname.com"`

## Startup:

Development:
> `npm run dev`

Production:
> `npm start`

## Commands:

Prefix:
> `~!`

Example command:
> `~!help`
