# Configuration

## Environmentals:

- `discord_token="" # your discord bot token`
- `discord_cmdPrefix="/plexcord" # defaults to ~!`
- `auth_password="password"`
- `plexLibrary="./com.plexapp.plugins.library.db"`
- `external_hostname="somehostname.com"`

## Startup:

Development:
> `npm run dev`

Production:
> `npm start`

## Commands:

Prefix:
> `~!`

Example command:
> `~! help`
